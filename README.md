## Assignment 1 – Back-End API with Node, Express & TypeScript

# This project is a simple API-oriented back-end built with **Node.js**, **Express**, and **TypeScript**. It includes:

- A basic API structure 
- A **health check** endpoint
- A **portfolio performance** endpoint that calculates investment performance
- Automated tests using **Jest** and **Supertest**
- **GitHub Actions** CI workflow for running tests and linting  push/PR
